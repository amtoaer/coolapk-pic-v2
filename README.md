# coolapk-pic

## Setup
```
git clone https://github.com/amtoaer/coolapk-pic-v2
cd coolapk-pic-v2
yarn
```

### Compiles and hot-reloads for development
```
yarn electron:serve
```

### Compiles and minifies for production
```
yarn electron:build
```
